## ✨ Fonctionnalités principales

**CyberTalent AI** connecte les talents cybersécurité aux recruteurs via :
- **CyberScore™** — score IA 0-100 basé sur certifications, labs, compétences et GitHub
- **Analyse IA** — résumé, points forts, axes d'amélioration, questions d'interview (Gemini + fallback local)
- **Matching intelligent** — recommandations d'emploi personnalisées
- **Dual-portal** — espace candidat (vert) et espace recruteur (rouge)
- **Administration** — panel de gestion complet

## 🚀 Démarrage rapide

```bash
git clone https://github.com/saotof-prog/cybertalent-ai.git
cd cybertalent-ai
npm install
# Configurer .env.local (voir README.md)
npx prisma db push
npm run dev
```
